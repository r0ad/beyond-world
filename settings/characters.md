# 角色设定

## 主角：林远（Lin Yuan）

### 基本信息
- **姓名**：林远
- **年龄**：28岁
- **性别**：男
- **职业/身份**：前世界交叉管理局（WCA）特工，现为流亡者
- **外貌特征**：中等身高，精瘦但结实，黑色短发，眼神深邃带着疲惫，左眉有一道疤痕
- **特殊标志**：右手腕有世界交叉点的能量纹身，是WCA特工的身份标识（已被部分抹除）

### 性格与心理
- **核心性格特质**：
  1. 坚韧不拔：即使面临绝境也不放弃
  2. 情感深沉：对爱人有着超越生死的执着
  3. 理性与感性的矛盾体：既能在关键时刻保持冷静，又会被情感驱动做出冒险决定
  4. 孤独：因为流亡身份而疏离人群
  5. 内疚：对自己过去的某些决定感到自责
- **内在动机**：找到并拯救爱人，弥补过去的遗憾，证明爱情可以跨越一切界限
- **价值观**：认为真正的爱情值得付出一切代价，即使与世界为敌
- **恐惧与弱点**：
  - 害怕再次失去爱人
  - 害怕自己的选择会伤害无辜的人
  - 对世界交叉点的能量有轻微恐惧（因为曾受过伤）
- **习惯与癖好**：
  - 习惯性地检查维度跳跃器的能量
  - 喜欢在安静的地方思考
  - 收集各个世界的小物件作为纪念
- **应对机制**：面对压力时会深呼吸，闭上眼睛回忆与爱人的美好时光

### 成长与发展
- **角色弧线**：从遵循规则的WCA特工，到违抗规则的流亡者，最终成为打破规则、重新定义可能性的英雄
- **转折点**：
  1. 爱人失踪时决定违抗命令私自寻找
  2. 被WCA通缉，成为流亡者
  3. 在世界尽头发现真相，面临最终选择
- **内在冲突**：在拯救爱人和维护世界秩序之间的道德挣扎
- **学习经验**：学会真正的爱情需要勇气，也需要智慧；规则不是绝对的，但代价必须承担

### 社交关系
- **家庭关系**：与家人关系疏远，因为长期执行任务很少回家
- **朋友/盟友**：
  - **艾米（Amy）**：自由联盟的技术专家，帮助林远穿越世界
  - **老陈**：前WCA同事，暗中提供帮助
- **敌人/对手**：
  - **WCA**：视他为通缉犯
  - **凯文·斯特朗**：WCA新任主管，严格执法者
- **恋爱关系**：
  - **苏雨（Su Yu）**：爱人，被困在世界尽头，是故事的核心
- **导师/学生**：无
- **其他重要关系**：与各个世界的居民有短暂但深刻的接触

### 背景故事
- **成长环境**：在主世界的一个普通家庭长大，父母都是普通工人
- **关键过去事件**：
  - 18岁加入WCA训练营
  - 25岁成为正式特工，执行跨世界任务
  - 26岁遇到苏雨，两人坠入爱河
  - 27岁，苏雨在调查世界尽头时失踪
- **创伤或挫折**：
  - 在一次任务中，队友因他的判断失误而牺牲
  - 苏雨失踪后，他违抗命令寻找，被WCA通缉
- **成就与胜利**：
  - 曾是WCA最年轻的优秀特工
  - 成功阻止过多次跨世界犯罪
- **隐藏的秘密**：他怀疑苏雨的失踪与WCA的高层有关，可能与世界尽头的秘密有关

### 能力与限制
- **特殊技能/天赋**：
  - 精通维度跳跃技术
  - 快速适应不同世界的物理规则
  - 优秀的战斗技巧和战术思维
  - 能够感知世界交叉点的能量波动
- **知识领域**：跨世界物理、世界交叉点理论、各世界文化
- **物理/心理限制**：
  - 频繁穿越导致身体对能量敏感，容易疲劳
  - 心理上对失去有强烈恐惧
- **资源与拥有物**：
  - 维度跳跃器（改装过，可以避开WCA监控）
  - 世界适应器
  - 能量护盾（已损坏，需要维修）
  - 苏雨的照片和笔记

### 故事功能
- **在故事中的角色定位**：主角，故事的推动者和视角人物
- **对主要情节的贡献**：穿越多个世界，解开谜团，最终到达世界尽头拯救爱人
- **与主题的联系**：体现"爱情可以跨越一切界限"的主题，证明真爱的力量

### 表现细节
- **独特的说话方式**：简洁直接，偶尔会引用苏雨说过的话
- **典型的肢体语言**：习惯性地摸手腕上的纹身，紧张时会握拳
- **着装风格**：实用主义，深色系，便于行动和隐藏
- **决策模式**：先理性分析，但最终会被情感驱动

### 发展规划
- **首次登场表现**：在一家小酒馆里，独自一人，收到关于苏雨的消息
- **主要成长节点**：
  1. 决定违抗WCA，成为流亡者
  2. 在某个世界遇到困难，学会接受帮助
  3. 发现真相，面临道德选择
  4. 在世界尽头做出最终决定
- **结局状态**：与苏雨团聚，但代价是改变了对世界的认知

## 女主角：苏雨（Su Yu）

### 基本信息
- **姓名**：苏雨
- **年龄**：26岁
- **性别**：女
- **职业/身份**：世界交叉点研究员，WCA科学顾问
- **外貌特征**：清秀的面容，长发及肩，眼神温柔但坚定，身材娇小但充满活力
- **特殊标志**：左手无名指有能量环，是她研究世界交叉点时获得的特殊标记

### 性格与心理
- **核心性格特质**：
  1. 聪明睿智：对世界交叉点有深刻理解
  2. 勇敢：敢于探索未知，即使知道危险
  3. 善良：关心所有世界的居民
  4. 独立：有自己的想法，不盲从权威
  5. 坚韧：即使被困也能保持希望
- **内在动机**：理解世界交叉点的本质，保护所有世界，与爱人团聚
- **价值观**：科学真理、世界和平、真爱至上
- **恐惧与弱点**：
  - 害怕孤独，害怕永远被困
  - 担心自己的发现会带来灾难
- **习惯与癖好**：
  - 喜欢记录一切，有详细的笔记习惯
  - 喜欢观察不同世界的星空
  - 习惯性地思考科学问题
- **应对机制**：通过科学研究和记录来保持理智

### 成长与发展
- **角色弧线**：从单纯的研究者，到发现真相的觉醒者，最终成为拯救世界的关键人物
- **转折点**：
  1. 发现世界尽头的秘密
  2. 被困在世界尽头
  3. 与林远重逢，共同面对最终选择
- **内在冲突**：在保护世界和维护个人感情之间的选择
- **学习经验**：真正的科学需要勇气，真爱需要牺牲

### 社交关系
- **家庭关系**：与父母关系良好，但父母不知道她的真实处境
- **朋友/盟友**：
  - **林远**：爱人，也是她最大的支持
  - **研究团队**：WCA的科学团队
- **敌人/对手**：
  - **WCA高层**：可能隐瞒了世界尽头的真相
- **恋爱关系**：与林远深爱彼此
- **导师/学生**：曾是某位著名科学家的学生

### 背景故事
- **成长环境**：知识分子家庭，从小对科学有兴趣
- **关键过去事件**：
  - 18岁进入WCA科学学院
  - 24岁成为世界交叉点研究专家
  - 26岁与林远相遇并相恋
  - 26岁在调查世界尽头时失踪
- **创伤或挫折**：被困在世界尽头，与外界失去联系
- **成就与胜利**：多项关于世界交叉点的重大发现
- **隐藏的秘密**：她发现了世界尽头的真实面貌，这可能改变一切

### 能力与限制
- **特殊技能/天赋**：
  - 顶尖的世界交叉点研究能力
  - 快速理解不同世界的物理规则
  - 能够感知能量流动
  - 优秀的逻辑推理能力
- **知识领域**：跨世界物理、能量学、维度理论
- **物理/心理限制**：
  - 被困在世界尽头，身体受到能量影响
  - 长时间的孤独可能影响心理状态
- **资源与拥有物**：
  - 研究笔记（记录了世界尽头的秘密）
  - 能量环（保护她免受世界尽头能量的伤害）

### 故事功能
- **在故事中的角色定位**：女主角，故事的"麦高芬"（主角寻找的目标），也是真相的持有者
- **对主要情节的贡献**：她的发现和选择将决定世界的命运
- **与主题的联系**：体现"翻越过世界来爱你"的主题，是爱情的对象和动力

### 表现细节
- **独特的说话方式**：会使用科学术语，但也会用诗意的语言描述
- **典型的肢体语言**：习惯性地推眼镜（即使没戴），思考时会用手指轻敲
- **着装风格**：实用但优雅，喜欢蓝色
- **决策模式**：基于科学和逻辑，但也会考虑情感因素

### 发展规划
- **首次登场表现**：在林远的回忆中，以及被困在世界尽头的场景
- **主要成长节点**：
  1. 发现世界尽头的真相
  2. 决定是否告诉林远真相
  3. 与林远重逢
  4. 共同面对最终选择
- **结局状态**：与林远团聚，但面临新的挑战和选择

## 重要配角

### 艾米（Amy）- 自由联盟技术专家

**基本信息**：
- 年龄：25岁
- 性别：女
- 职业：黑客和技术专家
- 外貌：染成蓝色的短发，穿着朋克风格

**性格**：
- 聪明、叛逆、讲义气
- 痛恨WCA的严格管控
- 帮助林远是因为相信真爱

**角色功能**：
- 提供技术支持和信息
- 代表自由联盟的立场
- 在关键时刻提供帮助

### 凯文·斯特朗（Kevin Strong）- WCA主管

**基本信息**：
- 年龄：45岁
- 性别：男
- 职业：WCA新任主管
- 外貌：高大威严，总是穿着正装

**性格**：
- 严格、冷酷、规则至上
- 认为个人的情感不能凌驾于世界秩序之上
- 隐藏着关于世界尽头的秘密

**角色功能**：
- 主要反派/对立者
- 代表秩序和规则
- 可能知道苏雨失踪的真相

### 老陈（Old Chen）- 前WCA同事

**基本信息**：
- 年龄：55岁
- 性别：男
- 职业：前WCA特工，现为情报贩子
- 外貌：沧桑的面容，总是叼着烟

**性格**：
- 经验丰富、重情义
- 对WCA的官僚作风不满
- 暗中帮助林远

**角色功能**：
- 提供关键信息和帮助
- 代表"灰色地带"的立场
- 在故事中起到转折作用
